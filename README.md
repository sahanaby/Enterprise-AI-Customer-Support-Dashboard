# Enterprise AI Customer Support Dashboard

## Overview

The **Enterprise AI Customer Support Dashboard** is a Python-based system designed to simulate how organizations manage and monitor customer support tickets.
It helps track customer issues, analyze ticket status, monitor response times, and visualize support operations using charts and dashboards.

This project demonstrates a simple enterprise workflow used in customer service and helpdesk environments.

---

## Features

* Customer support ticket dataset management
* Dashboard metrics (Total, Open, Closed tickets)
* Ticket status and priority tracking
* Response time analysis
* Data visualization using charts
* Interactive dashboard visualization
* Ticket search by Ticket ID
* Ticket filtering by Status and Issue Type
